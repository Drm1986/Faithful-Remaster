Faithful Remaster v11.9.5 - recovery note
=========================================

Version 11.9.5 prevents further permanent cleanup deletion, but it cannot restore
files that v11.9.4 already deleted with Python's unlink operation.

For Dolphin texture dumps, the safest recovery is usually to:
1. Keep Dolphin texture dumping enabled.
2. Revisit the affected Xenoblade areas, menus, battles, and cutscenes.
3. Dolphin will dump the encountered textures again into the Dump folder.
4. Keep Safe Startup Blank-Dump Quarantine OFF while rebuilding the dump set.

Also check any backup, copied dump pack, Windows File History, Previous Versions,
or recovery software before writing large amounts of new data to the same drive.
Existing remastered files in the Load folder are not removed by this cleanup.

In v11.9.5:
- Dynamic EFB/post-processing candidates are skipped and kept in place.
- Safe Cleanup Now only quarantines fully transparent or near-solid-black files.
- Quarantined files are stored under the current profile's _cleanup_quarantine folder.
