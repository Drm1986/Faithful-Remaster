# Before / After Gallery

These screenshots were captured from real emulator sessions. No screenshots below are mockups.

Texture replacement can improve texture clarity, UI artwork, backgrounds, logos, and character materials. Results vary by game, source texture quality, workflow settings, and emulator behavior.

> Some pairs may contain minor animation, camera, or notification differences. The comparison focus is texture quality.

## Notes

- The project aims to preserve the original artistic direction rather than redesign textures.
- The strongest gains usually appear in UI artwork, logos, background art, and very low-resolution source textures.
- Results depend on the selected ComfyUI workflow and processing settings.
