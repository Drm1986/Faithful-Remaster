# Contributing

Bug reports and reproducible emulator-specific findings are welcome.

Please include:

- Faithful Remaster version
- Emulator and version
- Game ID / Title ID
- Relevant Dump and Load folder structure
- Faithful Remaster logs
- Whether the issue affects RGB, alpha, or both
- A small non-copyright-sensitive test texture when possible

Do not upload ROMs, ISOs, copyrighted texture packs, or commercial model files.
