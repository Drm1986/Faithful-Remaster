Faithful Remaster v11.10.2 performance notes
============================================

- No additional full-resolution duplicate is kept during normal RGB processing.
- Workflow graph validation is cached naturally by file/process flow and is small compared with model inference.
- Output and Alpha integrity checks sample at most 128x128 pixels for similarity analysis.
- Hash cache remains optional and avoids repeated backend jobs for identical source/route combinations.
- Comfy input uploads reuse one remote filename per app thread, reducing Comfy input-folder growth.
- Texture Manager scanning and quarantine scanning remain background operations.
- Dynamic EFB protection samples images rather than running AI inference.
- VRAM protection remains disabled by default on all profiles, per requested build behavior.

PASS: bulk quarantine writes the manifest every 250 moved files instead of once per file.
PASS: live candidates are held outside processing queues until one bulk flush.

Profile migration runs once during startup and performs only small JSON/path synchronization; texture scanning and processing performance are unchanged.

UI note: v11.10.2 replaces six large dashboard toggles with four compact visible rows and one popup menu; processing performance is unchanged.
