# Faithful Remaster v11.10.25 Release Notes

v11.10.25 fixes Texture Manager live refresh selection behavior.

## Fixed

- Texture Manager no longer jumps back to the first texture after automatic refresh.
- Selected texture path(s) are restored after list rebuilds when still visible.
- Scroll position/viewport neighborhood is preserved when the selected texture is no longer visible.

## Not changed

- No bundled workflow JSON files were changed.
- No ComfyUI workflow behavior was changed.
- Sparse alpha/mask quarantine from v11.10.24 is preserved.
